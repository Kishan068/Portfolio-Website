Bouncy Bird

  • (Unity Play)

Game Engine: Unity
Programming Language: C#
Genre: 2D Casual GameBouncy Bird is a 2D casual game inspired by Flappy Bird, created to explore Unity's physics system. I enhanced the visuals with HD bird sprites and added a character selection feature with four different birds. The game uses Rigidbody physics for smooth movement and jump mechanics. Backgrounds, pipes, and coins are generated dynamically and destroyed behind the camera to optimize memory usage. Pipes and coins spawn independently at random positions to increase difficulty.I implemented a score system for both coin collection and pipe navigation, along with bird and coin animations for a more engaging experience. The game features increasing difficulty, a pause option, and a Game Over screen with restart and bird selection options. This project was designed to deepen my understanding of Unity’s physics system, including collisions, gravity, and dynamic object management. While not intended for commercial release, it demonstrates my ability to replicate and enhance classic game mechanics with modern polish.

Zombie Shooter

  • (Unity Play)

Game Engine: Unity
Programming Language: C#
Genre: 2D Casual GameZombie Shooter is a 2D casual game inspired by Stupid Zombies, developed to explore Unity’s animation system, AI behaviors, and physics-based shooting mechanics. The game features two levels with unique backgrounds and platform layouts. Zombies are placed at various positions and follow simple AI behavior to patrol platforms without falling. Each zombie can be in an idle or walking state. When hit by a bullet, a death animation plays, removing them from the scene.I implemented a gun that rotates toward the mouse pointer, with a real-time aim line rendered for precise shooting. Bullets are physics-based and bounce off screen edges, staying active for a short time before disappearing. Each level has a bullet limit, adding a layer of strategy to the gameplay. Players receive a star rating (1 to 3 stars) based on how efficiently they complete a level. For example, eliminating all zombies with a single bullet awards 3 stars. The game transitions between Home, Level Select, Gameplay, Game Over, and Victory screens for a complete user experience.This project helped me strengthen my skills in Rigidbody physics, collision handling, AI logic, and real-time line rendering in Unity.

Robotic Spider Model

3D Modeling Software: Autodesk Maya
Techniques: Polygon Modeling, Rigging, Animation
Genre: 3D Sci-Fi CreatureAs part of a Udemy course, I designed and developed a Robotic Spider Model using Autodesk Maya. This was my first full project in Maya, and it served as a great learning experience in 3D modeling, rigging, and animation.
I used polygon modeling to construct the spider's body, head, and legs, focusing on clean topology and an optimized mesh structure. Through this process, I significantly improved my understanding of working with vertices, edges, and faces.
To give the spider its mechanical and sci-fi appearance, I applied metallic shaders and materials inspired by futuristic designs. My goal was to visually communicate the robotic nature of the creature.
For rigging, I strategically placed joints to simulate realistic movement and used IK handles to enable smooth, natural leg motion. I used the Paint Skin Weights Tool to manually refine how the mesh deforms with each joint movement, ensuring precise and natural articulation of the spider’s legs and body. I also created individual rig controllers for each leg and body section, along with a main root controller for overall movement control.
I created three distinct animations, Idle 1, Idle 2, and Walk; using the rig and root controllers. These animations showcase the flexibility and realism of the spider’s motion.
This project helped me build a strong foundation in Autodesk Maya, including the interface, polygon modeling techniques, texturing, rigging, IK setup, and animation. It stands as a key portfolio piece demonstrating my ability to bring a sci-fi creature to life in 3D.
